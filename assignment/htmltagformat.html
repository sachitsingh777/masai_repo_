b tag

the b HTML element is used for reader attention to the element's content,which is not give a special attention , only used for draw a text in boldface. this b html element is know as boldface

i tag

the i HTML element is used for reader attention to the element's content, which some text change into special italicized type

u tag

The u HTML element represents a span of inline text which should be rendered in a way that indicates that it has a non-textual annotation. This is rendered by default as a simple solid underline, but may be altered using CSS

select tag

the select html element is used to create a menu list, the select element used in form , to collect user input.

button

the button html element is used to create a button which is clickable on web page , to complete a task.

hr

the hr HTML element is used for thematic break between paragraph level element or is used to separate content on web page

br

the br HTML element is used for producing single line break, which is used for poem, writing or an address
